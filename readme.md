# Scripts for development tools installation 

## V1
- OS
    - only Ubuntu on Raspberry 4b

## How to use
1. **Download repo**

    ```
    git clone https://github.com/DennisLiu16/pkg_installation_script.git
    ```

2. Find script named **Ubuntu_packages.sh**

3. Execute the script

    ```
    ./Ubuntu_packages.sh
    ```
